# splash_solution

## 🎯 功能说明
Splash 动态渲染方案集成，包含 Scrapy + Splash 的集成示例。

## 📌 技术栈
- Splash
- Scrapy
- Lua 脚本

## 📁 目录结构
```
splash_solution/
├── taobao_test/ # 淘宝测试
├── test_taobao_cookie/ # Cookie测试
└── test.lua # Lua脚本
```

## 🚀 快速开始
```bash
docker run -p8050:8050 scrapinghub/splash
# 然后运行 Scrapy 项目
```
