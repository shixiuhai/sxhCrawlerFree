# pinduoduo

## 🎯 功能说明
拼多多爬虫，覆盖商品搜索和详情页数据采集，涉及 anti_content 参数逆向。

## 📌 技术栈
- Python requests
- JS逆向 (anti_content)
- Playwright 自动化

## 📁 目录结构
```
pinduoduo/
```

## ⚠️ 稳定性风险
| 风险等级 | 说明 |
|----------|------|
| 🔴 HIGH_RISK | 拼多多反爬机制频繁更新，接口随时失效 |

## 🚀 快速开始
```bash
pip install requests playwright
playwright install chromium
```
