# baidu_qrlogin_drission

## 🎯 功能说明
基于 DrissionPage 实现百度账号扫码登录的演示代码。

## 📌 技术栈
- DrissionPage
- Selenium
- 百度扫码登录

## 📁 目录结构
```
baidu_qrlogin_drission/
├── demo.py
├── demo_api.py
├── extension/   # 浏览器插件
└── requirements.txt
```

## 🚀 快速开始
```bash
pip install DrissionPage
python baidu_qrlogin_drission/demo.py
```
