  • 伊凡·伊里奇之死

    9.1

    [俄] 列夫·托尔斯泰 / 东方出版社 / 2017-10

    列夫·托尔斯泰中短篇小说选，包括两个短篇《克罗采奏鸣曲》《魔鬼》和一个中篇《伊凡·伊里奇之死》，完美展现托尔斯泰风格魅力。

  • 豆树青青

    [美] 芭芭拉·金索沃 / 南海出版公司 / 2017-10

    《毒木圣经》作者芭芭拉·金索沃成名作，一部关于爱、友谊与成长之书，美国图书馆协会年度最佳青少年小说。

  • 命运的内核

    [英] 格雷厄姆·格林 / 江苏凤凰文艺出版社 / 2017-11

    格雷厄姆·格林公认的至高杰作，是格林对命运的探讨，穷尽了一个人在命运中会经历的一切考验。

  • 传家之物

  • 豹变

  • 少年们想从侧面看烟花

    [日] 岩井俊二 / 浙江文艺出版社 / 2017-10

    岩井俊二作品继《情书》《关于莉莉周的一切》之后又一部青春物语，电影《烟花》原著小说，完整收录1993年电影未透露细节。

    7.5

    [美] 菲利普·迪克 / 译林出版社 / 2017-10

    现象级畅销书《相约星期二》作者米奇·阿尔博姆新作，六根蓝色琴弦，六次人生的转折，将虚构的人物嵌入真实的美国音乐史。

  • 芬妮·希尔

    英国文学史上第一部真正意义的情色小说，欧洲「卖淫黄金年代」里妓女生活的缩影，侧面展现18世纪上半叶伦敦的社会风貌和生活图景。

  • 流吧！我的眼泪

    8.6

    [美] 菲利普·迪克 / 译林出版社 / 2017-10

    美国科幻小说家菲利普•迪克的代表作，坎贝尔奖最佳长篇小说。小说描绘了一种“平行世界”的可能，主人公在原来的世界里是举世瞩目的大明星，在“平行世界”里却默默无闻。

  • 海上钢琴师

    9.5

    [意大利] 亚历山德罗·巴里科 / 浦睿文化·湖南文艺出版社 / 2017-10
