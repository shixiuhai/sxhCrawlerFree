# selenium_wrapper

## 🎯 功能说明
Selenium Chrome自动化封装库，提供反检测浏览器配置和常用自动化操作封装。

## 📌 技术栈
- Selenium
- Chrome DevTools Protocol
- stealth.min.js 反检测

## 📁 目录结构
```
selenium_wrapper/
├── selenium_chrome_api/
│   └── stealth.min.js   # 反检测JS文件
└── ...
```

## 🚀 快速开始
```bash
pip install selenium webdriver-manager
python -m selenium_wrapper.selenium_chrome_api
```
