# taobao_cookie_sel

## 🎯 功能说明
基于 Selenium 获取淘宝登录 Cookie 的尝试，支持淘宝和1688平台。

## 📌 技术栈
- Selenium
- Chrome WebDriver
- Cookie 管理

## 📁 目录结构
```
taobao_cookie_sel/
├── test_selenium_getcookies.py
├── test_selenium_addcookies.py
├── login_taobao.py
└── ...
```

## 🚀 快速开始
```bash
pip install selenium webdriver-manager
python taobao_cookie_sel/login_taobao.py
```
